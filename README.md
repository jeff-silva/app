# Laravel + Nuxt + Docker
## Start dev server
```bash
docker-compose up
```

or use short command
```bash
yarn dev
```

## Start prod server
```bash
docker-compose -f docker-compose-prod.yml up
```

or use short command
```bash
yarn build
```
